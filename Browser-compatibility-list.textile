*Test method:*
The compatibility test is made in a native operating system, opening the "blueprint test page":http://blueprintcss.org/tests/ and each of the child pages in a browser, then checking that each page displays correctly. 

*Contributing:*
You can improve this list by repeating and validating the tests, testing a browser that's not available here, or if you have a more recent version of a browser. Test it, edit this page if you have a github account, and tell us on the "mailing list":http://groups.google.com/group/blueprintcss that you have an update.

*Bug/issue reports:*
If you have a browser issue, please make sure that you have tested it with the method outlined above. If you think the test pages should be improved,  please report it on the "bug tracker":http://blueprintcss.lighthouseapp.com/.
